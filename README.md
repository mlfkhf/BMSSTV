# BMSSTV

一个方便、简洁的音乐SSTV图片生成器！
如果你喜欢请你**帮忙点——亮右上角的Star** :star: 谢谢喵`

## `0x00` 特点

- **完全**跨平台，且**无需任何项目构建操作**！*一份代码，到处编译*。 :heavy_check_mark:<br><sub>注意：因为某些库使用了旧标准不支持的语法，你需要在C++20或以上语言标准下编译</sub>

- 没有繁琐的操作，**只需要一份MIDI文件**:musical_score::musical_keyboard:，就能把音乐“藏”在图片中。

- 支持多种SSTV模式，Martin1、Martin2、Scottie1、Scottie2、ScottieDX，以后还会有更多！ :radio:

- **控制台**程序！:scroll: 如果你喜欢用键盘而不是鼠标来处理事情，这对你来说刚刚好！

## `0x01` 如何使用？
别问我，问程序。
