# BMSSTV

A convenient and concise music-to-SSTV image generator!  
If you like it, please **⭐ Star** the repo in the top-right corner. Thank you, meow~ 🐱

## `0x00` Features

- **Fully** cross-platform, with **zero build configuration required**! *One codebase, compile anywhere*. :heavy_check_mark:  
  <sub>Note: Some libraries use syntax not supported by older standards, so you need to compile with C++20 or later.</sub>

- No tedious steps — **just one MIDI file** :musical_score: :musical_keyboard: is all you need to "hide" music in an image.

- Supports multiple SSTV modes: Martin1, Martin2, Scottie1, Scottie2, ScottieDX, with more to come! :radio:

- A **console** program! :scroll: If you prefer using the keyboard over the mouse, this is just right for you.

## `0x01` How to Use?

Don't ask me — ask the program.
